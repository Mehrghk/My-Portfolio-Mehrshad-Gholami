Bright Expo competition first challenge

The challenge was:

A simple encryption called the 'Caesar cipher' was used to send secret messages. This method works by shifting each letter in the message by a fixed number of positions in the alphabet. For example, if the letter 'A' is shifted 3 positions, it becomes 'D'. This encryption technique allows you to decode encrypted messages by applying specific algorithms.

And we had to design a program that could turn the encrypted letter into plain text.

This project is the first question of the France 2024 competitions. To use this program, you write a sentence in the first part and by specifying a certain number, you can move the letters of the sentence forward or backward and play with the letters. You can even use negative numbers.
